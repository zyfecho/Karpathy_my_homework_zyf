# 使用文档

## 1. 环境准备

### 1.1 安装 Python

需要 Python 3.10 或更高版本。验证安装：

```bash
python --version
```

### 1.2 安装依赖

```bash
cd C:\Users\12086\Desktop\my-knowledge-base
python -m pip install -r requirements.txt
```

### 1.3 配置 API Key

复制配置文件并填入你的 API Key：

```bash
copy .env.example .env
```

编辑 `.env` 文件：

```
OPENAI_API_KEY=sk-你的API密钥
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_MODEL=gpt-4o
```

如果使用中转站，将 `OPENAI_BASE_URL` 改为中转站地址，`OPENAI_MODEL` 改为中转站支持的模型名。

## 2. 快速开始

```bash
# 1. 查看原始资料解析状态
python app.py ingest

# 2. 检查知识库健康状态
python app.py lint

# 3. 向知识库提问
python app.py query "SPI和I2C有什么区别"

# 4. 导出为 HTML
python app.py export --format html
```

## 3. 命令详细用法

### 3.1 ingest — 查看原始资料

```bash
python app.py ingest
```

以表格形式展示 raw/ 目录下所有文件的名称、格式、大小和是否可解析。
支持的格式：PDF、Word(.docx)、PowerPoint(.pptx)、HTML、Markdown。

不需要 API Key，可直接运行。

### 3.2 compile — 编译知识库

全量编译（读取全部 raw/ 资料，重新生成 wiki/）：

```bash
python app.py compile
```

增量编译（只处理新增的资料文件）：

```bash
python app.py compile --incremental
```

需要 API Key。编译前建议备份现有 wiki：

```bash
xcopy wiki wiki_backup /E /I
```

### 3.3 query — 单次问答

```bash
python app.py query "你的问题"
```

示例：

```bash
python app.py query "SPI和I2C有什么区别"
python app.py query "STM32定时器怎么配置100ms中断"
python app.py query "FreeRTOS的任务间通信有哪些方式"
```

回答基于 wiki/ 中的全部内容，会引用具体来源文件。问答记录自动保存到 `outputs/sessions/`。

### 3.4 chat — 交互式多轮对话

```bash
python app.py chat
```

进入交互模式后：
- 直接输入问题，按回车发送
- 支持多轮追问，LLM 会记住上下文
- 输入 `quit`、`exit` 或 `q` 退出
- 退出时自动保存对话记录

### 3.5 lint — 健康检查

```bash
python app.py lint
```

检查 wiki/ 中的问题：
- 死链：`[[链接]]` 指向不存在的页面
- 孤立页面：没有被其他页面引用
- 空页面：内容过少
- Frontmatter 缺失：缺少 title/date/tags/sources

不需要 API Key。

### 3.6 export — 导出

导出为多个 HTML 文件（每个 wiki 页面一个）：

```bash
python app.py export --format html
```

导出为单个 HTML 文件（全部页面合并）：

```bash
python app.py export --format single
```

导出结果在 `outputs/` 目录下，可用浏览器直接打开。

## 4. 添加新资料

1. 将新文件放入 `raw/` 目录
2. 运行 `python app.py ingest` 确认文件可解析
3. 运行 `python app.py compile --incremental` 增量编译
4. 运行 `python app.py lint` 检查是否有问题

## 5. Obsidian 集成

1. 下载安装 [Obsidian](https://obsidian.md/)
2. 打开 Obsidian → "打开文件夹作为仓库" → 选择 `wiki/` 目录
3. 即可使用：
   - 图谱视图：查看所有页面的链接关系
   - 反向链接：查看哪些页面引用了当前页面
   - 双向链接跳转：点击 `[[链接]]` 直接跳转
   - Markdown 实时渲染

图谱视图入口：左侧边栏点击"关系图谱"图标，或使用快捷键 `Ctrl+G`。

## 6. 常见问题

**Q: 报错 `No module named 'rich'`**
A: 依赖没装到当前 Python 环境。运行 `python -m pip install -r requirements.txt`。

**Q: API 调用失败（503/429）**
A: 中转站暂时不可用或限流。等几分钟重试，或换一个模型名试试。

**Q: 编译后 wiki 页面质量不好**
A: 可以修改 `src/compiler.py` 中的 `COMPILATION_PROMPT`，调整提示词后重新编译。

**Q: 知识库太大超出上下文窗口**
A: 可以分主题放入不同子目录，分批编译。或使用支持更长上下文的模型。

**Q: 如何查看问答历史**
A: 所有问答记录保存在 `outputs/sessions/` 目录下，按时间戳命名。

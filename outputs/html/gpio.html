GPIO 输入输出

GPIO（General Purpose Input/Output）是STM32最基础的外设，用于控制引脚的高低电平输出和读取外部电平输入。

输出模式

推挽输出（Push-Pull）

  • 具有强驱动能力，可主动输出高电平和低电平
  • 一般不需要配置上下拉电阻（缺省模式即可）
  • 上拉+推挽：浪费电流，可能影响电平上升速度
  • 下拉+推挽：浪费电流，可能影响电平下降速度

开漏输出（Open-Drain）

  • 只能主动拉低，高电平依赖上拉电阻实现
  • 必须配合上拉电阻：GPIO_Mode = OUTPUT_OD; GPIO_Pull = GPIO_PULLUP
  • 典型应用：电平转换、[link] 通信

输出模式需要上下拉的特殊场景

  1. 开漏输出模式 — 必须配上拉
  2. 系统安全性要求极高 — MCU上电未初始化时，配置下拉可防止引脚意外输出高电平（如电机驱动、继电器）
  3. 硬件设计特殊要求 — 外部设备在MCU未初始化时需要确定的初始电平

初始电平配置

  • 推挽输出模式下，初始电平状态为高电平时默认灯灭
  • 建议不配置上下拉电阻（缺省模式）

HAL库常用函数

HAL_GPIO_WritePin(端口, 引脚号, 电平状态);   // 写引脚电平
HAL_GPIO_ReadPin(端口, 引脚号);             // 读引脚电平
HAL_GPIO_TogglePin(端口, 引脚号);           // 翻转引脚电平

中断回调

HAL_GPIO_EXTI_Callback(uint16_t 引脚号);    // 外部中断回调

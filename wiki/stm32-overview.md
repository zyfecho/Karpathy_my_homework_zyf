# STM32 微控制器概述

STM32是意法半导体（ST）基于ARM Cortex-M内核的32位微控制器产品系列，广泛应用于嵌入式系统开发。该系列包含多个子系列，覆盖从基本型到互联型的不同应用需求。

## 产品命名规则

以 `STM32F103C8T6` 为例：

- **STM32** — 基于ARM的32位微控制器
- **F** — 通用类型
- **103** — 增强型（101=基本型，102=USB基本型，105/107=互联型）
- **C** — 48脚（T=36, R=64, V=100, Z=144）
- **8** — 64K字节闪存（4=16K, 6=32K, B=128K, C=256K, D=384K, E=512K）
- **T** — LQFP封装
- **6** — 工业级温度范围 -40°C~85°C

## 产品子系列

| 子系列 | 特点 |
|--------|------|
| STM32F101xx | 基本型 |
| STM32F102xx | USB基本型，USB 2.0全速设备 |
| STM32F103xx | 增强型，最常用 |
| STM32F105xx/107xx | 互联型，支持以太网、USB OTG |
| STM32F407xx | 高性能型，Cortex-M4内核，168MHz |

## 技术文档体系

- **数据手册（Datasheet）**：芯片选型用，包含引脚分配、电气特性、封装信息
- **参考手册（Reference Manual）**：编程设计用，包含功能模块内部结构、寄存器配置
- **编程手册**：Cortex-M内核、SysTick定时器和NVIC的详细说明

在Keil中可通过 HELP → Open Books Window 快速查阅参考手册和数据手册。

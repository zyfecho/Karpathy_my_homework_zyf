# 传感器与模块

嵌入式开发中常用的外围模块，包括无线通信模块、显示模块、电机驱动和蜂鸣器等。

## ESP01 WiFi 模块

ESP01 基于乐鑫 ESP8266 芯片，支持 IEEE802.11b/g/n，内置 TCP/IP 协议栈。

### 工作模式

| 模式 | 说明 | 用途 |
|------|------|------|
| AP 热点模式 | ESP01 作为热点，其他设备连接它 | 1对多连接 |
| STA 连接模式 | ESP01 连接路由器或远程服务器 | 联网通信 |
| AP + STA | 两种模式同时开启 | 混合场景 |

### 常用 AT 指令

| 指令 | 功能 |
|------|------|
| `AT` | 测试模块连接状态 |
| `ATE0` | 关闭命令回显 |
| `AT+CWMODE=2` | 开启AP模式 |
| `AT+CWMODE=3` | 开启STA+AP模式 |
| `AT+CWJAP="SSID","PWD"` | 连接WiFi热点 |
| `AT+CIPSTART="TCP","IP",Port` | 连接TCP服务器 |
| `AT+CIPMODE=1` | 设置透传模式 |
| `AT+CIPSEND` | 开始透传 |
| `+++`（无回车） | 退出透传模式 |

### 硬件连接

- 使用 STM32 的 PC6/PC7 引脚（串口6）
- CubeMX 中配置 USART6 即可通信
- 模块天线方向朝实验板外

## Emm42 V5.0 步进闭环驱动

张大头闭环伺服推出的步进电机驱动模块，支持多种控制方式。

### 控制方式

- **脉冲控制**：STM32/Arduino/51/PLC 均可接入，通过 PUL/DIR/EN 信号控制
- **通讯控制**：支持串口TTL、RS232、RS485、CAN总线
- 支持多机通讯及同步控制

### 适用场景

3D打印、写字机、雕刻机、PLC控制、机械臂、小车比赛等。

## OLED 显示模块

- 0.96寸单色OLED，128×64分辨率，驱动IC为SSD1306
- 通信方式：[[i2c]] 或 [[spi]]
- 驱动文件注意 `SSD1306_i2c.c` 中引脚宏定义与实际接线一致

## 蜂鸣器（PWM控制）

- 无源蜂鸣器通过 [[timer]] 的 PWM 输出控制
- 频率控制音调（如 C5 = 523.25Hz）
- 占空比控制音量（典型50%）
- 脉冲个数控制时长
- 可用于播放 MIDI 音乐
